The box-shadow: property is used to generate and/or cast shadows at the block level; below I will give you a brief explanation of the missing complements to generate a spectacular shadow.

  1. Horizontal "→←" displacement of the shadow, offset-x: Specifies the horizontal distance; positive(+) and negative(-) values are also used; if a "+" value is specified, the shadow will be seen on the right side of the box/content, and if a "-" value is specified, the shadow will be seen on the left side of the box/content.
  2. Vertical "↑↓" displacement of the shadow, offset-y: Specifies the vertical distance; negative(-) and positive(+) values are also used; if a "-" value is specified, the shadow will be seen above the box/content, and if a "+" value is specified, the shadow will be seen below the box/content.
  3. Blur radius, blur-radius: Using this value is optional. If we specify a value greater than 0, the blurrier it will be, making the shadow softer. And if we don't specify a value, it will be 0 by default and consequently the shadow will appear sharper, without any blurring.
  4. Propagation radius, spread-radius: Using this value is optional. You can also use positive (+) and negative (-) values here; if you specify a positive value "+", the shadow of the box will expand and increase in size; if you specify a negative value "-", the shadow will shrink and decrease in size. The default value is 0, so it will have no effect, and the shadow will have the same size as the element.
  5. Color: And finally, don't forget the color, since the appreciation of the shadow will depend on it; you can use colors such as Hexadecimal, Rgba, Rgb, Hsl. Choose to use the color in the way that seems most efficient to you, I leave it to your taste.
